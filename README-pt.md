# Lista de Tarefas em JavaScript

Este repositório contém um gerenciador de tarefas simples desenvolvido com **HTML, CSS e JavaScript**.  
As tarefas são armazenadas diretamente no navegador utilizando **localStorage**.

---

## Como Utilizar

1. Clone este repositório:
   ```bash
   git clone https://github.com/Lymm-08/to-do-list.git
   ```
2. Abra o arquivo `index.html` em seu navegador.
3. Adicione, conclua e exclua tarefas livremente!

---

## Funcionalidades

- Adicionar novas tarefas  
- Marcar tarefas como concluídas  
- Excluir tarefas  
- Persistência de dados no navegador via **localStorage**

---

## Estrutura do Projeto

- `index.html` — Interface principal da aplicação.  
- `style.css` — Estilos visuais da aplicação.  
- `script.js` — Lógica em JavaScript para manipulação das tarefas.  
- `img/` — Recursos visuais utilizados no projeto.
